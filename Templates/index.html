SVM

// Support Vector Machine

A supervised learning algorithm that constructs an optimal hyperplane in a high-dimensional feature space to maximally separate data classes. The kernel trick enables transformation into non-linear decision boundaries without explicit computation.

  • Finds the maximum-margin hyperplane that separates classes with the widest possible gap between support vectors.
  • Kernel functions (RBF, Polynomial, Sigmoid) map data to higher dimensions for non-linear classification.
  • Regularization parameter C controls the trade-off between margin width and classification error.
  • Highly effective in high-dimensional spaces, even when dimensions exceed sample count.
  • Memory-efficient: decision function uses only a subset of training points (support vectors).

R.FOREST

// Random Forest Classifier

An ensemble method that builds multiple decision trees during training and outputs the mode of their predictions. By combining weak learners with bootstrap aggregation and random feature selection, it achieves remarkable robustness and accuracy.

  • Bootstrap aggregation (bagging) trains each tree on a random subset of data with replacement to reduce variance.
  • Random feature selection at each split decorrelates trees, making the ensemble stronger than any single model.
  • Naturally resistant to overfitting — more trees generally improve generalization without variance increase.
  • Built-in feature importance scores quantify each predictor's contribution to the model's decisions.
  • Handles missing values gracefully and scales to very large datasets with parallel training.
